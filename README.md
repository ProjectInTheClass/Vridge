# Vridge 🌿🌿🌿
#### " 채식이 어렵다고요? 더 이상 혼자 채식하지 마세요! Vridge가 있으니까요! "
> Vridge는 채식을 처음 입문하는 채식주의자들을 위한 채식 식단 챌린지 플랫폼입니다. <br>
